# Descrição do produto

Desenvolvida pensando no conforto do seu cliente e do profissional, a nova Linea Essenziale chegou para impressionar

A cadeira hidráulica Linea Essenziale, possui encosto Reclinável, além de perfeita para corte, é muito útil para procedimentos de maquiagem, design de sobrancelhas, etc.
Com materiais de excelente qualidade, cada peça é produzida exclusivamente.
Sabe aquela cor que vai arrasar no seu espaço? Fala com a gente, a gente tem pra você.
Você escolhe a base hidráulica e já deixamos montada.
Para atender tranquilamente aos clientes que necessitam de maior segurança devido ao peso corporal, recomendamos o uso da base hidráulica REFORÇADA tamanho Jumbo, com capacidade para suportar 150 kg.
